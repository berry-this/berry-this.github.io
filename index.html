Key Features

Maximum Performance

Resource-intensive background processes, telemetry, and spyware modules have been removed to significantly improve system responsiveness.

Full Control

Windows Update components are disabled, Microsoft Defender removed. You decide when to update and what protection to use.

Universal Compatibility

Artificial installation restrictions have been lifted. The system can be installed on PCs without TPM 2.0 and Secure Boot support.

Privacy by Default

Microsoft Edge has been completely removed. Installation is only possible with a local account, without needing a Microsoft account.

Post-Install Helper

Includes a post-install script for optional deep telemetry cleaning, classic context menu tweaks, and quick download of popular browsers and software.

Compact & Optimized

The size of the installed system is significantly reduced by removing dozens of unnecessary UWP apps and components.
